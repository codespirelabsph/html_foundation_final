Understanding Javascript

by Dinky Caorte

What is JavaScript?

JavaScript, often abbreviated as JS, is a programming language and core technology of the World Wide Web, alongside HTML and CSS. About 99% of websites use JavaScript on the client side for webpage behavior, often incorporating third-party libraries. All major web browsers have a dedicated JavaScript engine to execute the code on users' devices.

JavaScript is a high-level, often just-in-time compiled language that conforms to the ECMAScript standard. It features dynamic typing, prototype-based object-orientation, and first-class functions. It supports event-driven, functional, and imperative programming styles and provides APIs for working with text, dates, regular expressions, standard data structures, and the Document Object Model (DOM).

The ECMAScript standard itself does not include input/output (I/O) capabilities like networking or graphics. These are provided by the browser or runtime environment. JavaScript engines, originally used only in browsers, are now widely embedded in servers and apps — with Node.js being a popular example.

Client-side Usage

JavaScript is the dominant client-side scripting language on the Web. It allows scripts to be embedded in or included from HTML documents, enabling dynamic behavior and interaction with the DOM. Nearly all websites rely on JavaScript to enhance user experience and functionality.

Modern web browsers include built-in JavaScript engines that execute scripts directly on the user's device, contributing to responsive and interactive web pages.

Examples of Scripted Behavior

  • Loading content without reloading the page (e.g., via Ajax or WebSockets).
  • Animating webpage elements — such as fading, resizing, or moving objects.
  • Controlling media playback directly in the browser.
  • Generating alert boxes or pop-up messages.
  • Validating form input before submission to the server.
  • Tracking and logging user behavior for analytics and personalization.
  • Redirecting users to different pages or sections.
  • Storing and retrieving data locally using Web Storage or IndexedDB.
  • Enabling in-browser games or complex user interactions.

JavaScript Libraries and Frameworks

JavaScript's capabilities are extended by a wide variety of libraries and frameworks, which simplify complex tasks and accelerate development. These include tools like React, Angular, and Vue. While they are critical to modern development, they are discussed in detail in a separate section on Web Frameworks.
